### Диаграмма компонентов

<img width="373" height="635" alt="image" src="https://github.com/user-attachments/assets/32583e11-2aa1-438e-a757-1626434cd9d7" />

**Makefile** — автоматизирует сборку проекта.  
**bin/green_app** — итоговый скомпилированный исполняемый файл программы.  
**includes/db.h** — заголовочный файл с объявлениями функций для работы с базой данных.  
**src/main.c и database.c** — исходный код программы: логика работы и реализация функций.  
**green.db** — база данных SQLite, где хранятся все таблицы и информация приложения.
